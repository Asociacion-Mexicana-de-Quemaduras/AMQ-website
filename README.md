# Asociación Mexicana de Quemaduras — Sitio Web

Sitio web oficial de la **Asociación Mexicana de Quemaduras A.C. (AMQ)**, construido con Next.js como migración del sitio original en Wix.

La AMQ es una organización dedicada a promover la atención integral del paciente quemado a través de la investigación, educación y capacitación de profesionales de la salud en México. El sitio sirve como plataforma informativa para socios, profesionales de la salud y participantes del congreso anual.

## Páginas

| Ruta | Descripción |
|------|-------------|
| `/` | Inicio — hero del congreso, mensaje de bienvenida del presidente, paneles de secciones |
| `/nosotros` | Historia de la asociación y mesa directiva 2024–2026 |
| `/socios` | Información de membresía, cuota anual y datos de pago |
| `/congreso` | 44° Congreso Internacional 2026 — costos, programa y hospedaje |
| `/congreso/inscribete` | Formulario de registro al congreso |
| `/congreso/programa` | Programa preliminar del congreso |
| `/congreso/convocatoria` | Convocatoria para presentación de trabajos científicos |
| `/eventos` | Eventos y actividades de la asociación |
| `/contacto` | Información de contacto |

## Stack

- **Next.js 16** — App Router
- **React 19**
- **TypeScript**
- **Tailwind CSS v4**

## Desarrollo

```bash
npm install
npm run dev      # Servidor en http://localhost:3000
npm run build    # Build de producción
npm run lint     # ESLint
```
